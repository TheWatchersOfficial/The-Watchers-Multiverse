## 📖 关于作品

当全球治理组织被寡头架空，当超能力（GNOSIS）如随机诅咒般降临，一群游走在灰色地带的年轻人组建了 "The Watchers"。

从破败的赛博都市 Sphereberg，到冰封的南极基地；从异星花园 Zholo6561，到横跨光年的星际联盟；从对抗生存机器 E.R.O 的血战，到撕裂多元宇宙的史诗交锋…… 

这不仅是一场关于存续与自由的无尽跋涉，更是一次对“人类何以为人”的终极解码。

### 方式一：通过 GitHub 提交收录申请（推荐）
1. 确保您拥有一个 GitHub 账号（免费注册即可）。
2. 点击本页面上方的 **Issues**（议题）标签页。
3. 点击绿色的 **New issue**（新建议题）按钮。
4. 在标题中填写：`【二创收录】作品类型 - 作品名称`（例如：`【二创收录】插画 - ZEV·Prime飞行概念图`）。
5. 在正文处提供以下信息：
   - **作品名称**：
   - **创作者署名**：（您希望展示的名字）
   - **作品类型**：（小说/插画/设定/其他）
   - **作品下载链接/在线预览链接**：（请将文件上传至百度网盘/蓝奏云/阿里云盘等，并将分享链接贴在此处；如果是插画，也可直接粘贴图片直链）
   - **一句话简介**：（可选）
6. 点击 **Submit new issue** 提交。

我会在空闲碎片时间查看 Issues，下载您的作品并上传至 `/fanworks` 对应的文件夹中，并在提交信息中注明您的署名。收录完成后，我会关闭该 Issue。

### 方式二：通过社交媒体提交
如果您不习惯使用 GitHub，也可以在 [Bilibili、百度贴吧、小红书等平台] 发布作品时带上话题 **#TheWatchers开源共创#**，并通过私信或邮箱 [thewatchers@foxmail.com](mailto:thewatchers@foxmail.com)将作品文件发送给我。我会在有空时将其收录至仓库。
